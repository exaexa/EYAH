\exercise{}{Length}{Rewrite function \texttt{length} using list comprehension
and function \texttt{sum}.}

\exercise{}{Vowels}{Discard all non-vowel characters in a string.}

\exercise{}{Reasonable words}{From a list of words select only those that do
not have more than $\frac{1}{3}$ of letters `a' in all letters.}

\exercise{}{Whitespace}{From a list of numbers create a list of whitespace
strings of corresponding lengths:

\texttt{makews [1,3,0]}

should produce

\texttt{["\ ", "\ \ \ ", ""]}.}
