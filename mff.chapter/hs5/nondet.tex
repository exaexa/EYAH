\exercise{level=1}{Pairs}{%
Does a list generated by

\texttt{[(a,b) | a <- [1..], b <- [1..]]}

contain all pairs of natural numbers?

Generate a list where all pairs can be expected.
}
\exercise{level=1}{Pythagorean triples}{
Use list comprehensions to implement an infinite list that contains all
Pythagorean triples.
}
\exercise{level=3}{Pythagorean program}{
Reimplement the previous exercise using \texttt{do}-notation.

For that purpose, implement your own \texttt{guard}:

\texttt{guard :: Bool -> [()]}
}
